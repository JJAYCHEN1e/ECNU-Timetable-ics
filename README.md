# ECNU-Timetable-ics

This project is based on [Swift Package Manager](https://swift.org/package-manager/).

## Deploy


### Install Swift

Follow the instruction in [swift.org](https://swift.org/getting-started/).

### Build

`swift build`

### Generate Xcode Project

`swift package generate-xcodeproj`
